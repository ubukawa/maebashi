# maebashi
small scale data conversion


## install
```console
git clone https://github.com/ubukawa/maebashi
cd maebashi
npm install
vi config/default.hjson //edit config info, e.g. host, dbUser, dbPassword, etc.
```

## Possible workflow
run "work_small.sh" to creat small scale vector tile
